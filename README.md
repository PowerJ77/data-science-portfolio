# Work

## Agri-Vision US: Climate-Based Prediction of U.S. Corn Yield Using Machine Learning

This project explores how growing-season climate conditions influence county-level corn yield in the United States. I integrated USDA agricultural production statistics with PRISM climate observations and built machine learning models to estimate yield variability.

**Skills demonstrated**

- Data cleaning
- Feature engineering
- Machine learning
- Model evaluation
- Data visualization
- Reproducible workflow development

**Tools used**

- Python
- Pandas
- Scikit-learn
- XGBoost
- Matplotlib
- Jupyter Notebook

**Key result**

XGBoost produced the strongest performance among the tested models, with climate variables providing limited but measurable predictive power for yield.

Repository:  
https://github.com/PowerJ77/agri-vision-us
